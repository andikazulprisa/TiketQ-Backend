# TiketQ-Backend-API

TiketQ adalah RESTful API sederhana untuk manajemen tiket acara berbasis Flask dan SQLite.

## Fitur Utama

- CRUD Tiket
  - Buat tiket baru
  - Lihat semua tiket
  - Lihat tiket berdasarkan ID
  - Update data tiket
  - Hapus Tiket
- Validasi input menggunakan Marshmallow
- ORM: SQLAlchemy
- Database : SQLite

## Teknologi

- Python
- Flask
- SQLAlchemy
- Marsmellow
- SQlite

## Cara Menjalankan

1. **Clone Repository**

```bash
git clone https://github.com/andikazulprisa/TiketQ-Backend
cd TiketQ-Backend
```

2. **Aktifkan virtual environment (opsional)**

```
python -m venv venv
source venv/bin/activate    # MacOS/Linux
venv\Scripts\activate       # Windows
```

3. **Install Dependencies**

```
pip install -r requirements.txt
```

4. **Jalankan Aplikasi**

```
python app.py
```

5. **Cek di Postman**

```
http://localhost:5000/tickets
```
